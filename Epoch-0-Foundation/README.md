# EPOCH 0: The Foundation (Pre-Java)

## Mission Statement
*"What is a program? What is Java? Why does any of this matter?"*

Before you write your first line of code, you need to understand what code actually **is**. This Epoch demystifies the computer and programming itself.

---

## Lessons in This Epoch

1. **[Lesson 0.1: What is a Computer Program?](./Lesson-0.1-What-Is-A-Program.md)**
   - Understand what "code" really is (the recipe analogy)
   - Learn how computers "think"
   - Demystify programming before you start

2. **Lesson 0.2: Why Java?**
   - The "write once, run anywhere" story
   - Where Java is used in the real world
   - Why it's perfect for learning

3. **Lesson 0.3: Setting Up Your Workspace**
   - Installing the JDK (explained in human terms)
   - Choosing an IDE
   - Your first project setup

4. **Lesson 0.4: Your First "Hello, World"**
   - Breaking down every single character
   - Running your first program
   - Understanding what just happened

---

## Milestone for This Epoch

By the end of Epoch 0, you will:
- ✅ Understand that code is just instructions
- ✅ Know why Java exists and what makes it special
- ✅ Have a working development environment
- ✅ Have run your first Java program and understood every line

---

## Time Commitment
**Estimated**: 3-5 hours total (4 lessons)
